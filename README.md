## 목차
### 1. Intro
- Big-O 표기법
- 시간 복잡도와 공간 복잡도
- 간단한 문제 해결 패턴 소개
  - `sliding window`, `two pointer` ...
### 2. 재귀 (Recursion)
- base case
- 재귀 함수 패턴
- 자주 발생하는 에러
### 3. 탐색 알고리즘
- 이분 탐색 : O(N)
- 선형 탐색 : O(logN)
### 4. 정렬 알고리즘
- 기본 정렬 (비효율적 - O(N^2))
  - 버블
  - 선택
  - 삽입
- 심화 정렬 (효율적 - O(N logN))
  - 합병
  - 퀵
  - 기수 (Radix Sort)
