# 버블, 선택, 삽입 정렬 비교

## 버블 시간복잡도

- best : O(N)
- average : O(N^2)
- worst : O(N^2)
- 공간복잡도 : O(1)

## 삽입 시간복잡도

- best : O(N)
- average : O(N^2)
- worst : O(N^2)
- 공간복잡도 : O(1)

## 선택 시간복잡도

- best : O(N^2)
- average : O(N^2)
- worst : O(N^2)
- 공간복잡도 : O(1)

## 정리

- 버블, 선택, 삽입 정렬은 평균적으로 O(N^2)의 시간 복잡도를 가진다
- 데이터가 어느정도 정렬된 상황에서 버블, 삽입 정렬은 매우 빠르다
- 하지만 어느정도 정열된 상황에서도 선택 정렬은 빠르지 않다
